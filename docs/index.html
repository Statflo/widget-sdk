Demo overview:
This demo shows three seperate instances of a widget timer that generates a random number between 0 - 1000 and increments by one every second. The controls within the container allow you to select a widget, upon which an action shall occur (post, reset). For example, selecting the "one" radio button then pressing "reset" will set the time within the first widget to zero.

Similarily, when pressing a radio button within a widget - you are selecting the recipient widget to whom a postForward will be send.
